# Welcome to CO TRAVEL Web Application

## Project Info

CO TRAVEL is a modern, responsive travel planning web application built for hackathons and real-world demos.  
It helps users plan trips, manage itineraries, collaborate with groups, and visualize travel data through a clean UI.

---

## How can I edit this project?

There are several ways to edit and maintain this application.

### Use Your Preferred IDE (Recommended)

You can clone the repository and work locally.

#### Prerequisites
- Node.js
- npm (or yarn)

#### Steps

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd <YOUR_PROJECT_NAME>

# Step 3: Install dependencies
npm install

# Step 4: Start the development server
npm run dev
